HTML Codes and Descriptions

s
HTML Code Description
<!DOCTYPE html>Defines the document type and version of HTML.
<html>Root element of an HTML document.
<head>Container for metadata and links to resources.
<title>Specifies the document's title.
<body>Defines the document's body content.
<h1> to <h6>Defines headings, with <h1> being the largest.
<p>Defines a paragraph.
<br>Inserts a single line break.
<hr>Inserts a horizontal rule.
<a>Defines a hyperlink.
<img>Embeds an image.
<ul>Defines an unordered list.
<ol>Defines an ordered list.
<li>Defines a list item.
<table>Defines a table.
<tr>Defines a table row.
<td>Defines a table cell.
<th>Defines a table header cell.
<div>Defines a division or section.
<span>Defines an inline container.
<form>Defines a form for user input.
<input>Defines an input field.
<button>Defines a clickable button.
<label>Defines a label for an input element.
<select>Defines a dropdown list.
<option>Defines an option in a dropdown list.
<textarea>Defines a multi-line text input field.
<style>Defines CSS styles.
<script>Defines JavaScript code.
<link>Links to an external resource.
<meta>Provides metadata about the document.
<header>Defines a header for a section.
<footer>Defines a footer for a section.
<nav>Defines navigation links.
<article>Defines an independent piece of content.
<section>Defines a section of a document.
<aside>Defines content aside from the main content.
<main>Defines the main content of a document.
<em>Defines emphasized text.
<strong>Defines strong importance text.
<blockquote>Defines a block quote.
<code>Defines a piece of computer code.
<pre>Defines preformatted text.
<audio>Embeds audio content.
<video>Embeds video content.
<canvas>Defines a drawing area.
<svg>Defines scalable vector graphics.
<iframe>Embeds another HTML page.
